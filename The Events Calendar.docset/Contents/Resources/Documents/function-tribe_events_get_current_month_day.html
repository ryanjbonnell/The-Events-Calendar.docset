Function tribe_events_get_current_month_day

Used in the month loop. Gets the current day in the month loop

Returned array contains the following elements if the day is in the currently displaying month: 'daynum' => Day of the month (int) 'date' => Complete date (Y-m-d) 'events' => Object containing events on this day (WP_Query) 'total_events' => Number of events on this day (int) 'view_more' => Link to the single day (URL)

If the day is part of the previous or next month, the array simply contains: 'date' => 'previous' or 'next'
