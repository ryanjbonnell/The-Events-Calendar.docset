Function tribe_events_the_month_day

Used in the month loop. Advances the loop pointer to the next day, and sets that day up for use.

Example: < code > <?php // loop through the days in the current month query while( tribe_events_have_month_days() ) : tribe_events_the_month_day(); // do stuff endwhile; ?> </ code >
